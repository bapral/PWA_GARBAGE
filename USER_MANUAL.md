# 🚛 台灣垃圾車即時地圖：使用者操作手冊 (2026 版)

本應用程式整合了台灣五大直轄市（台北、新北、台中、台南、高雄）的開放資料，提供即時位置追蹤與智能班表預測功能。

---

## 📍 1. 快速上手：初次進入系統

當您第一次開啟應用程式時，系統會自動執行以下流程：
1.  **初始化同步**：系統會根據您所在的城市下載最新的垃圾清運班表至本地資料庫，確保在離線或 API 斷訊時仍能運作。
2.  **定位權限要求**：請允許應用程式存取「精確位置」，以便在地圖上顯示您的位置並搜尋最近的垃圾車。
3.  **預設畫面**：系統預設會顯示「新北市」的即時地圖。

---

## 🌆 2. 縣市切換：跨區查詢

您可以隨時切換到不同城市查看當地的清運動態：
*   **操作方式**：點擊畫面上方工具列（AppBar）中的 **「城市圖示」**（位於右上角倒數第二個）。
*   **支援縣市**：台北市、新北市、台中市、台南市、高雄市。
*   **自動調整**：切換後，地圖視角會自動移動到該城市的中心點，並重新下載對應的班表資料。

---

## 🧭 3. 定位模式：自動 vs 手動

本程式提供兩種定位模式，方便您靈活查詢：
*   **自動 GPS 定位（預設）**：地圖會跟隨您的實際位置。圖示顯示為紅色大頭針 📍。
*   **手動指定地點**：若您想查詢其他地點（如公司或親友家）附近的垃圾車。
    *   **切換方式**：點擊畫面上方標題文字（例如「新北市垃圾車即時地圖」）即可切換模式。
    *   **操作**：切換為手動模式後，在畫面上**任意點擊**，藍色標記 📍 就會出現在該處。

---

## 跑 4. 核心功能：搜尋最近的垃圾車

這是本程式最實用的功能，能幫助您快速決定何時出門：
1.  **點擊綠色按鈕**：按下右下角的 **「綠色雷達按鈕」**。
2.  **顯示導航線**：系統會在地圖上畫出一條綠色連線，指向距離您最近的車輛。
3.  **步行資訊**：畫面底部會顯示：
    *   目標車牌號碼。
    *   直線距離（公尺）。
    *   **預估步行時間**（以一般人步速計算）。

---

## 🔮 5. 預測模式：掌握未來動向

除了即時位置，您還可以「預見未來」：
*   **開啟方式**：點擊右上角的 **「時鐘圖示」**。
*   **相對時間預測**：例如設定「30 分鐘後」，查看屆時垃圾車預計會行經哪些路段。
*   **絕對時間查詢**：例如設定「今晚 20:30」，檢索該時段全市的班表分佈。
*   **回歸即時**：若要切換回當下的即時動態，點擊上方的 **「回到現在」** 卡片或刷新按鈕即可。

---

## ℹ️ 6. 車輛詳細資訊

點擊地圖上任何一個 **「垃圾車標記」**，會彈出詳細資訊面板：
*   **車牌號碼**：確認車輛識別。
*   **清運路線**：顯示該車所屬的清運路線名稱。
*   **狀態與時間**：
    *   如果是 **即時模式**，會顯示「GPS 最後更新時間」。
    *   如果是 **預測模式**，會顯示「資料庫預定抵達時間」。
*   **距離估計**：再次顯示該車與您的距離與步行建議。

---

## 🛠️ 7. 進階維護與疑難排解

*   **強制更新資料**：若發現班表過舊，可點擊右上角的三個點 `⋮` 選單，選擇 **「強制清除並更新資料庫」**。
*   **重新整理**：在即時模式下，點擊 AppBar 的 **「刷新圖示」** 可立即抓取政府 API 的最新經緯度。
*   **顏色識別**：不同城市的車輛標記有不同外框（北：藍、新：黃、中：綠、南：橘、高：紫），方便辨識。

---

## 🌐 PWA (Web 版) 特殊說明

若您透過瀏覽器（網頁版）使用：
*   **離線支援**：支援 PWA 技術，安裝到手機桌面後，即使在收訊不良處也能讀取已快取的班表。
*   **代理伺服器**：為了繞過瀏覽器限制，Web 版會自動切換多個 Proxy 代理，確保資料抓取不中斷。

---
*資料正確性以各地方環保局官網班表為準，GPS 訊號受環境影響可能會有 30-60 秒延遲。*
